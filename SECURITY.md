# Security Policy

## Supported Versions

The following table outlines the support status for security updates. The current active version is `3.0.0`. We do not plan to update older versions. When version `4.0.0` is released, support for `3.0.x` will end.

| Version | Supported          |
| ------- | ------------------ |
| 3.0.x   | :white_check_mark: |
| < 3.0   | :x:                |

## Reporting a Vulnerability

To report a security vulnerability, please do not use public GitHub issues. Instead, please report it by creating [a new draft security advisory](https://github.com/danielklecha/SharpIppNext/security/advisories/new).

### What to expect

- **Updates**: We will keep you informed of the progress as we investigate and address the issue.
- **Coordinated Disclosure**: We will work with you to coordinate a release that fixes the vulnerability before making details public.
